# `videos/` — Vídeos do site

Vídeos curtos usados no site (fundos, destaques).

## Convenção de nomes

Minúsculas com `-`, ex.: `cabedelo-amanhecer.mp4`.

## Recomendações

- Formato **`.mp4`** (H.264) ou **`.webm`**, para compatibilidade.
- Para fundos: sem som, curtos (< 20 s), comprimidos (idealmente < 5 MB).
- Resolução 1080p chega para a maioria dos usos web.
- Para vídeos longos/pesados, considera alojar no YouTube/Vimeo ou Supabase Storage
  em vez de os meter no repositório.
