# `imagens/agenda/` — Capas dos eventos

Foto de capa de cada evento da agenda.

## Convenção de nomes

Usa o **mesmo slug** do evento. Exemplo:

- Evento `open-de-surf-cabedelo-2026` → `imagens/agenda/open-de-surf-cabedelo-2026.jpg`

No evento, define:

```
cover_url = /imagens/agenda/open-de-surf-cabedelo-2026.jpg
```

## Recomendações

- Proporção **16:9** (ex.: 1600×900 px).
- `.webp` ou `.jpg`, < 400 KB.
